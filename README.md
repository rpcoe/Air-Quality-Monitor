



Open the folder in VScode. When the PicoW starts it runs the program in code.py. To enable quickly changing programs, I use the 'import xxx' statement, which will run the program xxx.py . Ignore VScode warnings about imports not being resolved. The imports will be pulled from the 'lib' folder on the Pico
Every time a change is made to xxx.py (or code.py) the program automatically restarts.
Open a serial monitor terminal to see the program output.  
A web browser can open the server by loading the IP address of the PicoW or by using the url: 192.168.4.1 on a browser connected to the PicoW through wifi.???
